# getModInformation()

Makes a request to the given mod URL, downloads its information table and returns an object with the information parsed.

If the first request fails, it will attemp five more times with different delays depending on the situation.

After exausting all tries, if nothing works, it throws an exception with the problem found.

Returns

Void

# requestHtml(theURL, onResponseFn, onResponseParams[, currentTry, self]) private method

Makes a GET request to the given URL to download its HTML content and on successfull completion of the request it executes the given function with the given parameters.

If the first request fails, it will attemp five more times with different delays depending on the situation.

After exausting all tries, if nothing works, it throws an exception with the problem found.

Parameters

theURL string

The URL of the page whose HTML content we want to download.

onResponseFn function

The function to be executed once the request successfully completes.

onResponseParams Array

The arguments of the onResponseFn.

currentTry number Optional

The current amount of tries. Used to keep track of how many times we made the request to the server in case of error.

self ModScraper Optional

Reference to the context object used to make the subsequent calls if the first request fails.

Returns

Void
